# TRAFFIC DEVILS TEST TASK

## Як запустити вручну
### Фронтенд

```bash
# Перейдіть до папки client:
cd client

# Встановіть залежності:
npm ci

# Створіть файл .env на основі .env.example:
cp .env.example .env

# Запустіть розробку:
npm run dev
```

### Бекенд

```bash
# Перейдіть до папки server:
cd server

# Встановіть залежності:
npm ci

# Створіть файл .env на основі .env.example:
cp .env.example .env

# Запустіть розробку:
npm run dev

```
